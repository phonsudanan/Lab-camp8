CSS syntax ประกอบด้วย การเขียน CSS Selector Combinator

selector{property:value}

Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น <table> ก็จะแสดงผลในส่วนของตาราง

Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ชนิดตัวอักษร , ขนาดตัวอักษร

Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , ตัวอักษรแบบ tahoma , ตัวอักษรขนาด 12 pt

table{border:1px solid #C00;}
Selector คือ table หมายถึง ต้องการกำหนดลักษณะให้กับแท็ก table
Property คือ border หมายถึง สิ่งที่ต้องการกำหนดให้กับ table นั่นก็คือ ขอบ(border)
Value คือ 1px หมายถึง 1px มีลักษณะเป็นเส้นเดียว solid มีสีเป็น C00 สีแดงเข้ม ๆ

Inline Style Sheet การเขียนแทรกลงไปในบรรทัดนั้นๆ ของ tag HTML เลย

Embed Style Sheet การเขียนแบบ Internal (embedded) คือเขียนแยก CSS ออกจาก tag HTML แต่อยู่ในหน้าเดียวกันกับไฟล์ HTML โดยแทรก css เอาไว้ที่ <head></head>

External Style Sheet การเขียน แยก css ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้ใน หน้าเว็บด้วย การเรียกโดย ใช้ โดยการพิมพ์ <link rel="stylesheet" href="externalcss.css" type="text/css" /< ไว้ที่ส่วน <head></head>

ID Selectors : การกำหนดชื่อของ element โดยจะมีได้แค่ element เดียวเท่านั้น เขียนได้โดยการเขียนชื่อ Selector ตามหลังเครื่องหมายชาร์ป (#)

Class Selectors : การกำหนดชื่อของ element โดยสามารถมีได้หลาย element เขียนได้โดยการ ตั้งชื่อ Selector ไว้หลังเครื่องหมายจุด หรือ Dot (.)

Pseudo element : การกำหนดเฉพาะบางส่วนของ element

Pseudo class : การกำหนดเฉพาะบางสถานะของ class

Combinators : การกำหนดเงื่อนระหว่าง 2 selector

Attribute selector : การกำหนดการเลือกชื่อ attribute ใน element

descendant selector (space)
a b = tag b ทั้งหมดที่อยู่ใน a

child selector (>)
a > b = tag b ที่เป็นลูกของ a

adjacent sibling selector (+)
a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น

general sibling selector (~)
a ~ b = tag b ที่อยู่ล่าง a ทุก tag เช่น h1~p { } คือ ให้กระทำการใดๆ กับ p ที่อยู่หลัง h1 ไม่ว่าจะอยู่ในลำดับที่เท่าไร
