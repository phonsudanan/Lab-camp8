CSS syntax ประกอบด้วยอะไรบ้าง

selector property value
Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น <table> ก็จะแสดงผลในส่วนของตาราง Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ชนิดตัวอักษร , ขนาดตัวอักษร Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , ตัวอักษรแบบ tahoma , ตัวอักษรขนาด 12 pt

การเขียน CSS มีกี่แบบอะไรบ้าง

Inline Style Sheet Embed Style Sheet External Style Sheet
การเขียนแทรกลงไปในบรรทัดนั้นๆ ของ tag HTML เลย การเขียนแบบ Internal (embedded) คือเขียนแยก CSS ออกจาก tag HTML แต่อยู่ในหน้าเดียวกันกับไฟล์ HTML โดยแทรก css เอาไว้ที่ <head></head> การเขียน แยก css ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้ใน หน้าเว็บด้วย การเรียกโดย ใช้ โดยการพิมพ์ <link rel="stylesheet" href="externalcss.css" type="text/css" /< ไว้ที่ส่วน <head></head>

selector มีกี่แบบอะไรบ้าง

ID Selectors Class Selectors TAG Selectors
การกำหนดชื่อของ element โดยจะมีได้แค่ element เดียวเท่านั้น สามารถเขียนได้โดยการเขียนชื่อ Selector ตามหลังเครื่องหมายชาร์ป (#) การกำหนดชื่อของ element โดยสามารถมีได้หลาย element เขียนได้โดยการ ตั้งชื่อ Selector ไว้หลังเครื่องหมายจุด หรือ Dot (.) TAG Selectors อ้างอิงกับชื่อ Tag ของ HTML โดยตรง เช่น <p> หมายความว่า ถ้าหากกำหนดคุณสมบัติให้แก่ TAG Selectors นี้แล้ว Tag ของ HTML ที่มีชื่อเดียวกับ Selectors นี้ จะมีคุณสมบัติของ CSS เหมือนกันหมดทุก Tag ภายในเว็บเพจนั้น การเขียน TAG Selectors เราสามารถเขียนได้โดยการเขียนชื่อ TAG ของ HTML ลงไปโดยตรงหน้าเครื่องหมายปีกกา { ..... }

combinator มีกี่แบบอะไรบ้าง

descendant selector (space) child selector (>) adjacent sibling selector (+) general sibling selector (~)
a b = tag b ทั้งหมดที่อยู่ใน a a > b = tag b ที่เป็นลูกของ a a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น a ~ b = tag b ที่อยู่ล่าง a ทุก tag เช่น h1~p { } คือ ให้กระทำการใดๆ กับ p ที่อยู่หลัง h1 ไม่ว่าจะอยู่ในลำดับที่เท่าไร

adjacent sibling selector (+)

ให้กระทำการใดๆ กับ section ที่อยู่ต่อจาก section ที่เป็นพี่เท่านั้น

ul + p {
color: red;
}
