position z-index

Property position ใน CSS นั้น จะมี value ดังนี้

1.static นั้นถือเป็นค่าเริ่มต้นสำหรับ position

2.relative แสดงผลต่อจาก ณ จุดที่มันอยู่ตรงนั้น “แบบตรงไปตรงมา” ซึ่ง position: relative; มีลักษณะคล้าย ๆ กับ static ต่างกันตรงที่ relative นั้น เราสามารถใช้ top left bottom right ในการกำหนดตำแหน่งได้

3.absolute จะแสดงผลเป็น “อิสระ” คือจะให้มันไปอยู่ตรงไหนก็ได้ ซึ่งมันจะอยู่แค่ภายใต้ element ที่ครอบมันอีกทีเท่านั้น

4.fixed ทำหน้าที่คล้าย absolute เลย แต่มีแตกต่างกันที่ fixed นั้นจะใช้กับ relative เพื่อกำหนดกรอบของมันไม่ได้ และจะตรึงตัวเองอยู่ตรงนั้นเสมอ

5.sticky การรวม relative กับ fixed ไว้ด้วยกัน โดยการกำหนด relative เพิ่มว่าจะให้ไป fix ที่ top หรือ bottom ทำให้เราเห็น block นั้นเลื่อนและถูกตรึงไว้เมื่อเรา เลื่อน scroll เมาส์

z-index คือการใช้จัดเรียง element ที่ซ้อนกันและต้องเป็น element ที่ถูกกำหนด position ไว้เท่านั้น ตัวเลขยิ่งมากก็จะอยู่บนสุด

sample position

div 1
position: fixed จะตรึงตัวเองอยู่ตรงนั้นเสมอ
div 2
position: static เป็นค่าเริ่มต้นสำหรับ positionเลื่อนตามหน้าเว็บปกติ
div 3
position: relative แสดงผลต่อจาก ณ จุดที่มันอยู่ตรงนั้น
div 4
position: absolute จะแสดงผลเป็น “อิสระ”
div 5
position: sticky การรวม relative กับ fixed ไว้ด้วยกัน โดยการกำหนด relative เพิ่มว่าจะให้ไป fix ที่ top หรือ bottom ทำให้เราเห็น block นั้นเลื่อนและถูกตรีงไว้เมื่อเรา เลื่อน scroll เมาส์
